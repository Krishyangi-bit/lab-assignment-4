# Prompt Quizzer – JavaScript Quiz Game

## Description
This project is a basic JavaScript quiz game that runs in the browser console.
It asks the user questions using `prompt()` and shows feedback using `alert()`.

The quiz uses basic JavaScript concepts such as arrays, loops, functions,
conditional statements, and string methods.

---

## Features
- Predefined quiz questions
- User input using prompt()
- Case-insensitive answer checking
- Score tracking
- Final score display

---

## Concepts Used
- Variables (var)
- Arrays
- Objects
- for loop
- if-else condition
- Functions
- prompt() and alert()
- toLowerCase() and trim()

---

## How to Run
1. Open your browser
2. Open Developer Tools → Console
3. Paste the code from `quiz.js`
4. Press Enter
5. Answer the questions shown in prompts
